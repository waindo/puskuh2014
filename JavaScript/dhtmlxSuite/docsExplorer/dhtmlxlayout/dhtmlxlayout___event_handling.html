Event Handling

The following events are available in dhtmlxLayout:

  • onDock/onUndock Events - occurs when the user clicks the left mouse button on item's button “Dock”/“Undock”;
  • ResizeFinish Event - occurs when the user finishes resizing the Layout;
  • onPanelResizeFinish Event - occurs when the user finishes resizing an item;
  • onExpand Event - occurs when an item is being expanded;
  • onCollapse Event - occurs when an item is being collapsed.

Attaching Event Handler

The user can add any user-defined handler to any of the available events.

To do this the user can use attachEvent() method with the following parameters:

  • evName - name of the event;
  • evHandler - user-defined event handler.
    <script>
        dhxLayout.attachEvent(evName, evHandler);
    </script>

Note: the names of the events are case-insensitive.
