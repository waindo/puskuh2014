dhtmlxLayout

dhtmlxLayout is a component that provides an easy way to compose several items into a single desktop layout. This is a very effective way of placing different types of content on one page. The component is flexible and easy to use. Different types of predefined layout patterns are designed to help users to save time on configuring.

This component operates such notion as “Items”. Items are parts of the Layout (in other words, cells of layout table) that can contain different types of content.

Main Features

  • Crossbrowser compatibility;
  • Single command integration with dhtmlx components;
  • Full Screen/Window modes;
  • Resizable items;
  • Collapsable items;
  • Wide range of predefined layouts;
  • Predefined skins, inherited by included components;
  • Script API.
